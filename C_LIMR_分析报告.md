## 功能概述

C_LIMR 是一个限幅器功能块，用于限制实数类型输入值的范围。该功能块将输入值限制在上限和下限之间，当输入值超出范围时，输出相应的限幅值。

## 流程路径描述

### 上限限幅路径：
开始 → 输入值 > 上限 → 输出上限值
**功能**: 限制输入值不超过上限

### 下限限幅路径：
开始 → 输入值 < 下限 → 输出下限值
**功能**: 限制输入值不低于下限

### 正常输出路径：
开始 → 下限 ≤ 输入值 ≤ 上限 → 输出输入值
**功能**: 输出正常范围内的输入值

## 逐帧功能分析

### Rung 7: 上限检测

**功能描述**: 检测输入值是否超过上限

**输入条件**:
| 信号名称 | 信号描述 | 信号类型 | 触发值 |
|----------|----------|----------|--------|
| X | 输入 | REAL | 数值 |
| UL | 上限值 | REAL | 设定值 |

**输出功能**:
| 信号名称 | 信号描述 | 信号类型 |
|----------|----------|----------|
| UL1 | 上限值1 | REAL |

**触发逻辑**:
- IF X > UL THEN UL1 = UL

**功能实现**: 
使用GT（大于）比较器，当输入X大于上限UL时，输出上限值UL到UL1。

### Rung 7: 下限检测

**功能描述**: 检测输入值是否低于下限

**输入条件**:
| 信号名称 | 信号描述 | 信号类型 | 触发值 |
|----------|----------|----------|--------|
| X | 输入 | REAL | 数值 |
| LL | 下限值 | REAL | 设定值 |

**输出功能**:
| 信号名称 | 信号描述 | 信号类型 |
|----------|----------|----------|
| LL1 | 下限值1 | REAL |

**触发逻辑**:
- IF X < LL THEN LL1 = LL

**功能实现**: 
使用LT（小于）比较器，当输入X小于下限LL时，输出下限值LL到LL1。

### Rung 8: 限幅输出

**功能描述**: 根据限幅条件输出相应值

**输入条件**:
| 信号名称 | 信号描述 | 信号类型 | 触发值 |
|----------|----------|----------|--------|
| X | 输入 | REAL | 数值 |
| UL | 上限值 | REAL | 设定值 |
| LL | 下限值 | REAL | 设定值 |

**输出功能**:
| 信号名称 | 信号描述 | 信号类型 |
|----------|----------|----------|
| Y | 输出 | REAL |

**触发逻辑**:
- IF X <= UL AND X >= LL THEN Y = X
- IF X > UL THEN Y = UL
- IF X < LL THEN Y = LL

**功能实现**: 
使用LE（小于等于）和GE（大于等于）比较器，当输入X在上下限之间时，输出X；当X超出范围时，输出相应的限幅值。

## 触发条件总结

### 限幅条件
- **上限限幅**: X > UL
- **下限限幅**: X < LL
- **正常输出**: LL <= X <= UL

## 实现功能总结

### 主要功能
1. **上限限幅**: 限制输入值不超过上限
2. **下限限幅**: 限制输入值不低于下限
3. **正常输出**: 输出正常范围内的输入值

## 调试技巧

### 调试步骤
1. 检查X值，确认输入正常
2. 检查UL和LL值，确认限幅范围设置正确
3. 监控Y值，观察限幅输出

### 常见问题
1. **限幅不工作**: 检查UL和LL值设置
2. **输出不正确**: 检查X值是否在限幅范围内
