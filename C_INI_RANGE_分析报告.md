## 功能概述

C_INI_RANGE 是一个范围检测功能块，用于检测整数值是否在指定范围内。

## 逐帧功能分析

### Rung 7: 范围检测

**功能描述**: 检测VAL是否在MAX和MIN范围内

**输入条件**:
| 信号名称 | 信号描述 | 信号类型 | 触发值 |
|----------|----------|----------|--------|
| VAL | 前值 | INT | 数值 |
| MAX | 最大值 | INT | 设定值 |
| MIN | 最小值 | INT | 设定值 |

**输出功能**:
| 信号名称 | 信号描述 | 信号类型 |
|----------|----------|----------|
| VAL | 前值 | INT |

**触发逻辑**:
- IF MIN <= VAL <= MAX THEN VAL保持不变
- ELSE VAL = 0

**功能实现**: 
使用LE和GE功能块，检测VAL是否在MIN和MAX范围内。

## 触发条件总结

### 范围条件
- **在范围内**: MIN <= VAL <= MAX
- **超出范围**: VAL < MIN OR VAL > MAX
